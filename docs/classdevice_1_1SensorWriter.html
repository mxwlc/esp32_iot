Detailed Description

Class represensting a Sensor Object
This is used to update the database as well as creating readings objects that keeps track of the MAC Address of the device the sensor is attached to aswell as the individual Sensor Address.
@var _sensor_address: Address of the sensor connected to a device
@var _device: Device object that the sensor is connected to
@var _sensor_name: Sensor name to identify object
@var _sensor_description: Brief description of sensor
@var _unit: Unit of readings
